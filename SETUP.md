# Challenge Setup

## All in one script

1. Get a Ubuntu VM
2. Connect to the machine with SSH
3. Clone this repo and run the script
```
 export GITHUB_USER=
 export GITHUB_TOKEN=
 git clone https://$GITHUB_USER:$GITHUB_TOKEN@github.com/papaux/dns-challenge.git
 cd dns-challenge
 ./ubuntu-setup.sh
```
